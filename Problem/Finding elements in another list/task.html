Finding elements in another list Open on JetBrains Academy

Write a program that finds the index of each element of one list in another one.

Input: the first line, as well as the second one, contains a list of integer numbers separated by space. The numbers of the second line are sorted in ascending order. The numbers of the second list are unique.

Output: for each element of the first list, find the index of the element in the second list.

If the target element is not present in the second list, return $-1$ as its index.


Sample Input:
3 1 2 5 4
1 2 3 4 5

Sample Output:
2 0 1 4 3



Memory limit: 256 MB
Time limit: 2 seconds
