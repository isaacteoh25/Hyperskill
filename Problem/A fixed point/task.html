A fixed point Open on JetBrains Academy

For a sorted list $L$ consisting only of unique numbers, a fixed point is an index $i$ such that $L[i] = i$. Write a program that identifies whether a sorted list contains a fixed point.

Input: a sorted list of unique integer numbers separated by spaces.

Output: True if the list contains a fixed point and False otherwise.

Note that the problem is easy to solve using linear search. We recommend you to solve this problem using binary search to improve the understanding of the algorithm.


Sample Input:
-8 -2 0 3 9

Sample Output:
True



Memory limit: 256 MB
Time limit: 15 seconds
