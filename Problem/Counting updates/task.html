Counting updates

For different lists, selection sort performs a different number of updates of the index in the inner for loop. index is the current index of the minimum element in the unsorted part of the list.

Write a program that for a collection of lists finds one for which selection sort performs the maximum number of such updates provided that the lists are expected to be sorted in ascending order.

Input: the first line contains the number of lists $n$. Each of the following $n$ lines contains a list of integer numbers separated by spaces.

Output: two numbers separated by spaces. The first one is the index of a list for which selection sort performs the maximum number of updates, the second one is the number of updates for this list. If there are several lists with the maximum number of updates, print the index of the first one.

As you iterate over the unsorted part of the list in the inner for loop, count how many times you encounter the smallest element that you've seen so far.


Sample Input:
3
1 2 4 3
3 4 2 1
5 4 3 1

Sample Output:
1 4



Memory limit: 256 MB
Time limit: 15 seconds
