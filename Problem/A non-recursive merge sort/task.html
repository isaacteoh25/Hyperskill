A non-recursive merge sort

The described merge sort algorithm is recursive: at each step, it splits an input list into two parts, recursively sorts them and then merges into one sorted list. However, there is a non-recursive version of the algorithm. Your task here is to implement such a version of merge sort.

Input: a list of integer numbers separated by spaces.

Output: the list of the same numbers sorted in descending order.

Note that the problem might be easily solved using either the described version of merge sort or the standard Python sorting utils. However, we encourage you to implement your own non-recursive version of merge sort.


Sample Input:
3 1 2 5 4

Sample Output:
5 4 3 2 1
