Hashtags

A hashtag is metadata used in social networks to easily tag a message with user-defined words. Hashtags start with # (the number sign, or octothorpe, whichever word you prefer). A user just places # in front of a particular word in the message. That helps other users to find this message along with other messages containing this word.

In this task, you should find all the words in a message that are hashtags. Try not to use standard Python string methods because we want you to practice algorithms. A hashtagged word starts with a single symbol # and ends with space or any of these punctuation marks .,!?. Also, a hashtagged word counts if it is the last word in a message, even though there is no trailing space or punctuation mark after it.

Print each word that matches these criteria on a new line.


Sample Input:
#How Long Does# It ###Take### #To #Learn #Python?

Sample Output:
How
To
Learn
Python

Time limit: 15 seconds
