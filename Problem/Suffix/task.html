Suffix Open on JetBrains Academy

Write a program that prints only filenames with .py extension.

Input: the first line contains the number of filenames n and is followed by n filenames, each on its own line.

Output: all the files with .py extension, listed on separate lines.

Checking whether a string ends with a specific substring is one of the most common tasks, which is why Python has a string method called str.endswith(). It is equally important to know about it and to be able to implement this algorithm yourself. You are supposed to write your own function here. Seize a chance to test your strength!


Sample Input:
6
__init__.py
main.css
readme.md
entropy
canopy
hyperskill.py

Sample Output:
__init__.py
hyperskill.py



Memory limit: 256 MB
Time limit: 15 seconds
