Up to permutation

Let's say that two strings are equal up to symbols permutation if we can change the order of symbols of the first string so that it equals the second one. For example, strings aabc and abca are equal up to symbols permutation because we can reorder the symbols of the first string and get the second one.

Write a program that identifies whether a text contains a substring equal to a pattern up to symbols permutation.

Input: two strings, the first one is the text, the other one is the pattern, each on a separate line.

Output: True if the text contains a substring equal to the pattern up to symbols permutations and False otherwise.


Sample Input:
abracadabra
aacd

Sample Output:
True



Memory limit: 256 MB
Time limit: 6 seconds
