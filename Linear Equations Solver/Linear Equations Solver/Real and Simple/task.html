Description

A linear equation is an equation of a straight line, often presented as \(y = a * x + b\). It is called linear because it contains only variables to the power of 1, where \(x\) and \(y\) are variables, and \(a\) and \(b\) are coefficients.

Linear equations are present in many fields of science and technology. A Sciencing article titled How Are Linear Equations Used in Everyday Life? can give you more detail on this. Over the course of this project, you are going to write quite a powerful linear solver, but for now, we will start with a simple one.

Write a program that solves \(a * x = b\).

Numbers \(a\) and \(b\) should be read from the first line of the standard input. You should output the value \(x\) so that the equation has a solution. For example, if the input contains the numbers \(5\) and \(3\), you should interpret it as \(5 * x = 3\). For this example, the answer would be \(0.6\), as \(5 * 0.6 = 3\).

Objectives

Write a program that solves \(a * x = b\) and then prints a solution.

In this stage, your program should:

  1. Read and parse the \(a\) and \(b\) coefficients from the user input.
  2. Solve the \(a * x = b\) equation.
  3. Output \(x\).

Example

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

> 5 3
0.6
