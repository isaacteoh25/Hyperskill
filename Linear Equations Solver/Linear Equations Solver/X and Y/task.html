Description

Our task is becoming more complicated! Now we have one additional variable (\(x\) and \(y\)) and one additional equation. In general, this equation would look like this:

\[\begin{cases} ax+by=c \\ dx + ey = f\end{cases}\]

where \(a, b, c, d, e, f\) are numbers. The general algorithm is to separate the variables so that the first variable is solved in the first line and the second variable is solved in the second line.

The main concept here is to subtract rows. You already know how to subtract numbers, and row subtraction is pretty similar. The example below shows the subtraction of the first row from the second one:

\[\begin{cases} ax+by = c \\ dx - ax + ey - by = f - c \end{cases}\]

As you can see, we subtract \(a * x + b * y\) from the left side of the second equation, we also subtract \(c\) from the right side of the equation. Since we subtracted equal amounts from both sides, the equation remains correct. You can also simplify it like this:

\[\begin{cases} ax+by = c \\ (d-a)x + (e-b)y = f - c \end{cases}\]

It all comes together when you realize you can subtract only a part of the equation. Let's subtract the first equation multiplied by \(d / a\) from the second equation. Here's the first step of this operation:

\[\begin{cases} ax+by=c \\ dx*d/a + ey*d/a = f*d/a\end{cases}\]And here's the result of this operation:

\[\begin{cases} ax+by = c \\ (d-a * d/a)x + (e-b * d/a)y = f - c*d/a \end{cases}\]

Notice that \(d - a * d / a\) becomes \(d - d\), which is \(0\). Therefore, we can exclude \(x\) from the second equation.

\[\begin{cases} ax+by = c \\ (e-b * d/a)y = f - c*d/a \end{cases}\]

At this stage, we can find the \(y\) value. Divide both sides by \(e - b * d / a\).

\[\begin{cases} ax+by = c \\ y = \dfrac{f - c*d/a}{(e-b * d/a)} \end{cases}\]

Now, you can find the \(x\) value from the first equation since you know the \(y\) value. This is similar to what we had in the previous stage.

The first line of the input contains the numbers \(a, b, c\). The second line of the input contains the numbers \(d, e, f\) numbers. All six numbers are floating-point numbers. Then, you should output the solution (\(x\) and \(y\)) in a single line.

The WolframAlpha site can help you check whether your solution is correct. Click the "Approximate form" button to get the decimal representation of simple fractions.

Objectives

Write a program that solves the following equations: \[\begin{cases} ax+by=c \\ dx + ey = f\end{cases}\], and prints \(x\) and \(y\).

In this stage, your program should:

  1. Take and parse the coefficients from the user's input that consists of six numbers, written in two lines.
  2. Solve the equation above to find \(x\) and \(y\).
  3. Output \(x\) and \(y\).
