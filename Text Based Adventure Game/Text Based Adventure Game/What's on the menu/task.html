Description

In this stage, you need to put up the main menu for the game.

Think about the video games you like. They may be different, but they have one thing in common â€” the main menu. It may contain the game title and some options like "start a new game" or "load game". And of course, it should have the option to exit the game.

Objectives

In this stage, your program should:

  1. Show a welcome message with the game title.
  2. Show different options to start a new game or load a saved one.
  3. Take the user input. If it is unknown, show a message and wait for the correct one.
  4. Print temporary placeholder messages like in the example, when a player starts a new game or loads an existing one.
  5. Implement the exit feature for your game. Print a goodbye message.

Your program must be able to process inputs of different letter cases.

Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

2- Press key '2' or type 'load' to load your progress
3- Press key '3' or type 'quit' to quit the game
