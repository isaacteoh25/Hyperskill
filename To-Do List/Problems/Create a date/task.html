Below we have a Date class. In its __init__ method we've specified two attributes: day and month.

class Date:

    def __init__(self, day, month):
        self.day = day
        self.month = month

On the basis of these attributes, we want to create a new one, a date that will represent day and month string values divided by a slash. For example, for 14 and 05 we should get 14/05.

This attribute could have been set in the initializer, too. In such a case, however, if we were to change the day value, say, to 15, we would still get 14/05. What we actually want to get is the following:

date_obj = Date(14, 05)
print(date_object.date)  # 14/05

date_obj.day = 15
print(date_object.date)  # 15/05

Write the body of the date method that will return the values of day and month separated by a slash and make it accessible as an attribute by placing the correct decorator name above. Note that you don't have to take input or create new class instances.

NOTE: in this case, both day and month are strings, so they can have leading zeros.
