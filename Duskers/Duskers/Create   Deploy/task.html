Description

We promised robots: it's time for robots! In this stage, you should create your own robots using websites such as fsymbols.com to help with the design. You will also add a menu to the game screen.

Objectives

  1. Your robots are represented by their ASCII/Unicode images, which can be up to 7 lines tall. In this stage, you should have three robots.
  2. Place the robots into the Hub you made earlier.
  3. Add the menu to our game screen. The menu should have the following options: back to game, return to main menu, save and exit, and exit. Make a nice little box around your menu like the one in the example.
  4. The menu options should do what they say, except for save and exit: just insert a placeholder message there and end the program. We will deal with saving later.

Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.
